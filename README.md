# WEAVE Public Docs

This repository is the clean public documentation surface for WEAVE.

It exists to publish approved public documentation without exposing the internal operating substrate, private drafting history, or internal mechanics documents from the private working repository.

- [Month 1 Overview](month1/README.md)

## Publishing boundary

This repo is intentionally narrow.

Included here:
- approved public Month 1 docs
- public lifecycle explanation

Not included here:
- internal operating substrate
- internal mechanics and settlement docs
- private drafting notes
- service-integration strategy
- private governance or operator procedures

## Source of truth

The canonical working source remains the private WEAVE repository. This public repo is a deliberate projection of approved public-safe docs only.
